===============
python-serpente
===============

Roman numeral conversion. Supports Python 3.6, 3.7, 3.8, and PyPy3.6.

.. image:: https://github.com/jcarbaugh/python-serpente/workflows/Python%20package/badge.svg

Usage
=====

    >>> import serpente
    >>> serpente.encode(1234)
    u'MCCXXXIV'
    >>> serpente.decode(u'MCCXXXIV')
    1234
