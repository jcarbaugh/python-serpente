===============
python-serpente
===============

Roman numeral conversion module.

.. image:: http://img.shields.io/travis-ci/jcarbaugh/python-serpente.png
   :target: https://travis-ci.org/jcarbaugh/python-serpente

Usage
=====

    >>> import serpente
    >>> serpente.encode(1234)
    u'MCCXXXIV'
    >>> serpente.decode(u'MCCXXXIV')
    1234
