# python-serpente

Roman numeral conversion. Supports Python 3.8 to 3.12.

![main branch CI badge](https://github.com/jcarbaugh/python-serpente/workflows/Python%20package/badge.svg)

## Usage

``` python
>>> import serpente
>>> serpente.encode(1234)
'MCCXXXIV'
>>> serpente.decode("MCCXXXIV")
1234
```
